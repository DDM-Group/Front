Создает оболочку базы ЦУНАМИ.
А.Как запустить:
1) Скачать пакет необходимых программ (WebStorm)
2) Клонируем репозиторий с GitHub в WebStorm (выбираем в окне старта вкладку GitHub и вставляем ссылку проекта)
3) Качаем Node.JS
4) Указываем путь к папке с Node.JS через Edit configurations...(если не указана автоматически)
5) Устанавливаем необходимые библиотеки через терминал: npm install
6) Запускаем сайт через терминал: npm start
7) Если сайт не запускается автоматически, перейти по ссылке http://localhost:3000
8) При ошибке выполнить рекомендации, указанные на странице и обновить библиотеки Node
